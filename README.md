The Site for flask-appbuilder
-----------------------------

## How to contribute:

* Create a virtual enviroment and activate it
* `pip install Flask-AppBuilder`
* `cd app`
* `export FLASK_ENV=development`
* `flask fab create-admin`
* `flask run`
